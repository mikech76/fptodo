Срок реализации 7 календарных дней (т.е. до 12.06.2017 включительно).

ЗАДАНИЕ:
Необходимо реализовать приложение - простой ToDo-список с сохранением в БД.
Задание охватывает следующие языки, утилиты и технологии: php, mysql, javascript, ajax, git.
Приложение в итоге должно выглядеть вот так: http://todomvc.com/examples/backbone/

Для реализации необходимо:
- клонировать (git clone) https://github.com/tastejs/todomvc-app-template - это шаблон приложения, т.е. уже готовая верстка;
- реализовать всю функциональность клиентской части приложения на javascript. Можно использовать библиотеки: jQuery, prototype и т.д. (использовать фреймверки, представленные на http://todomvc.com/, нельзя);
- реализовать сохранение текущего списка todo в базу данных MySQL при помощи ajax, бекенд - на php;
!- реализовать регистрацию, аутентификацию и авторизацию пользователя для того, чтобы пользователи могли создавать личные todo-листы.

Допы (это - необязательная часть задания):
!- реализовать возможность “расшаривания” todo-списков (в любом виде, как душа пожелает), т.е. пользователь A может “расшарить” свой todo-список с пользователем B;
!- реализовать кеширование и инвалидацию кеша todo-списокв;
- реализовать уровень доступа к расшаренному todo-списку: юзер может либо только читать, либо читать и редактировать;
- протестировать php модули при помощи phpunit.

Допы реализовывать не обязательно, они являются небольшим усложнением поставленной задачи и не влияют на общее представление о качестве кода, полученного в результате выполнения обязательной части задачи.
